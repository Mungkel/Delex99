Delex99 a 3D printed fully open source keyboard


This keyboard had the priority to be an allrounder

99% gasket mount southpaw with inversed numpad

inspired by CAD and Gaming

When I designed the board I thought about my habit to game, which preferred mousespace, well for my CAD usage I'd enjoy if I had a numpad. Due to these requirements I put the numpad on the left to making it southpaw, but for my muscle memory I thought it'd be best if the numpad was just like the left hand that now operates it mirrored to the right. Due to the fact, that I do not know how popular this is going to be I design no PCB, meaning this keyboard has to handwired.


The keyboard consist of as shown in the image the case and the split main and numpadplate and the magnetically attached topcase. The bottomcase and topcase get attached by either gluing 6x2mm round magnet in reversed polarity to both cases or with a CNC'd steel (or any otheer magnetic material) topcase with only one set of magnets.
